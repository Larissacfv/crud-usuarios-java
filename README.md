# crud-usuarios-java
Projeto CRUD em Java para cadastro, busca, edição e remoção de usuários, com armazenamento em arquivo .txt.
# CRUD de Usuários em Java

Projeto desenvolvido para praticar **Java**, **lógica de programação** e **Programação Orientada a Objetos (POO)**.

O sistema implementa um **CRUD (Create, Read, Update, Delete)** para gerenciamento de usuários, com persistência de dados em arquivo.

## Funcionalidades

- Cadastrar usuário
- Listar usuários
- Buscar usuário por ID
- Atualizar dados do usuário
- Deletar usuário

## Tecnologias utilizadas

- Java
- Programação Orientada a Objetos (POO)
- Manipulação de arquivos (.txt)

## Como funciona

Os dados dos usuários são armazenados em um arquivo chamado **cadastro.txt**.

Cada usuário possui:
- ID único
- Nome
- Idade
- Sexo

## Objetivo do projeto

Praticar a criação de um sistema CRUD em Java utilizando persistência de dados em arquivo e organização do código em classes.
